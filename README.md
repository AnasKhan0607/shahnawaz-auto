# Shahnawaz Auto Care Website

Professional website for Shahnawaz Auto Care, a family-owned auto repair shop in Etobicoke, Toronto.

## 🔧 About

Shahnawaz Auto Care provides honest, reliable auto repair services to the Etobicoke community. This website was built to establish their online presence and help customers find them easily.

**Location:** 1475 Albion Rd, Etobicoke, ON M9V 1B1  
**Phone:** (416) 742-9615

## 🌐 Live Site

[View Live Site](https://anaskhan0607.github.io/shahnawaz-auto/)

## ✨ Features

- Mobile-first responsive design
- Click-to-call functionality
- Google Maps integration
- Service listings
- Fast loading (no frameworks, pure HTML/CSS/JS)
- SEO optimized

## 🛠️ Tech Stack

- HTML5
- CSS3 (Custom properties, Grid, Flexbox)
- Vanilla JavaScript
- Google Fonts (Inter)

## 📱 Services Highlighted

- Oil Changes
- Brake Service
- Engine Repair
- A/C Service
- Battery & Electrical
- Safety Inspections
- General Repairs
- Tire Services
